# UI Aesthetics

Use this steering file for web interface polish and judgment.

## Core rules

- preserve scope and information density
- prioritize composition, spacing, and typography before decoration
- keep accents restrained and semantic
- distinguish resting, hover, pressed, focus, selected, loading, success, and error states
- treat shadow, blur, glow, and motion as structural tools, not spectacle
